Projeto de Demonstração Git Flow

Este projeto foi criado para demonstrar o uso do Git Flow no gerenciamento de ramificações.

Versão atual

v1.0.1

Branches utilizadas

  • main: versão oficial do projeto
  • develop: versão em desenvolvimento
  • feature: novas funcionalidades
  • release: preparação para lançamento
  • hotfix: correções urgentes

Histórico de versões

  • v1.0.0 - Lançamento inicial do projeto
  • v1.0.1 - Correção de bug na identificação da versão
